# Password Generator

A fast, secure, and fully customizable tool for generating strong, random passwords. Built with clean, modern frontend technologies, this application provides users with complete control over their security settings.

---

## Live Demo

[https://veon321.github.io/password-generator/](https://veon321.github.io/password-generator/)

---

## Technologies Used

- **HTML5:** Semantic structure for optimal accessibility.
- **CSS3:** Modern styling featuring Flexbox and responsive typography.
- **JavaScript (Vanilla JS):** Core logic for random character generation, DOM manipulation, and Clipboard API integration.
